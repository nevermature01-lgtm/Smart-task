# **App Name**: TaskMaster Pro

## Core Features:

- User Authentication: Secure user registration and login using Supabase Auth.
- Task Creation: Allow users to add new tasks with descriptions.
- Task Listing: Display tasks in a clear, responsive list.
- Task Editing: Enable users to edit task details.
- Task Completion: Allow users to mark tasks as complete.
- Real-time Updates: Update the task list in real-time using Supabase Realtime.
- Data Persistence: Store and manage tasks in Supabase PostgreSQL Database.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey a sense of focus and organization.
- Background color: Light gray (#EEEEEE), desaturated from the primary blue, to provide a clean and calm backdrop.
- Accent color: A slightly more saturated blue-purple (#5C6BC0) to give focus to interactive elements without pulling too much attention.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and highly readable interface.
- Use clear, simple icons to represent task actions and status.
- A clean, card-based layout that is intuitive and responsive across devices.
- Subtle animations for task creation, completion, and editing to enhance user experience.
